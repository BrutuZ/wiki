# Movedone groups
```
series:
  settings:
    group a:
      set: { movedone: /share/media/a/ }
    group b:
      set: { movedone: /share/media/b/ }

  group a:
    - series 1
    - series 2
  group b:
    - series 3
    - series 4
```

You can use this in feed or preset
